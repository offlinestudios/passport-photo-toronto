# Passport Photo Toronto Website Structure

## Navigation Structure
The website will have a consistent navigation menu across all pages with the following items:
- Home
- Services
- Pricing
- FAQ
- Location (added to all pages as requested)
- Contact
- Book Appointment (call-to-action button)

## Page Structure

### 1. Homepage (index.html)
- Header with logo, navigation menu, and call-to-action
- Hero section with main value proposition and call-to-action
- Our Services section with three categories (with proper icons):
  - Canadian Passport Photos
  - Visa & Immigration Photos (moved before ID & License)
  - ID & License Photos (consolidated with Medical & Driving Permit)
- Why Choose Us section with benefits
- Pricing overview with updated prices ($24.99)
- Customer testimonials
- Find Our Studio section with map
- Footer with updated information:
  - Address: 63 McCaul St. Toronto ON, M5Y 2W7
  - Phone: (416) 986-2677
  - Hours: Daily 12PM - 6PM
  - Copyright: © 2025 Passport Photo Toronto. All Rights Reserved.

### 2. Services Page (services.html)
- Header with logo and navigation
- Main services introduction
- Service categories in the following order:
  1. Passport Photo Services
  2. Visa & Immigration Photos
  3. ID & License Photos (consolidated with Medical & Driving Permit)
- Each category will have proper icons and links to individual service pages
- Our Photo Process section
- Compliance Guarantee section
- Call-to-action
- Footer

## Responsive Design
- The website will be fully responsive for all devices
- Mobile navigation will use a hamburger menu
- Images and layout will adjust appropriately for different screen sizes

- Proper meta titles and descriptions for all pages
- Semantic HTML structure
- Optimized image alt tags
- Schema markup for local business
- Breadcrumb navigation
- XML sitemap
- Robots.txt file
